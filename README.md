# Hytale Russian (RU)

![](https://minecraft.fr/wp-content/uploads/2026/01/ce-que-la-communaute-doit-comprendre-du-lore-de-hytale.webp)

Русификатор для игры **Hytale**

Этот мод добавляет русский язык в Hytale, переводя интерфейс, меню и игровые элементы.  
Проект не является официальным и создаётся фанатами игры.

## ⚡️ Установка (Windows)

1. Откройте лаунчер Hytale  
2. Перейдите в **настройки** (иконка шестерёнки)  
3. Нажмите **«Open directory»**  
4. Скопируйте папку **`install`** в любое другое место (на случай восстановления)  
5. Распакуйте архив с модом (`Hytale-Russian_Vx.x.x.zip`)  
6. Перенесите папку **`install`** из архива в папку игры  
7. Подтвердите замену файлов  

Готово! Запустите игру — интерфейс будет на русском языке.

---

## 🍎 Установка на macOS

> На macOS структура игры немного отличается.

1. Откройте лаунчер **Hytale**  
2. Перейдите в **настройки** (иконка шестерёнки)  
3. Нажмите **«Open directory»** — откроется папка `Client`  
4. Найдите файл **`Hytale.app`**  
5. Нажмите по нему **ПКМ → Show Package Contents**  
6. Перейдите по пути:  `Contents → Resources → Data → Shared`
7. Распакуйте архив с модом (`Hytale-Russian_Vx.x.x.zip`) 
8. Скопируйте содержимое:
- папку **`Fonts`** → в `Shared/Fonts`
- папку **`Language`** → в `Shared/Language`
9. Подтвердите замену файлов, если система запросит

Готово! Запустите игру — интерфейс будет на русском языке.

## ❎ Восстановление оригинальных файлов

Если нужно вернуть оригинальную версию игры:

1. Откройте лаунчер Hytale  
2. Перейдите в **настройки** → **«Open directory»**  
3. Удалите текущую папку **`install`**  
4. Верните ранее сохранённую оригинальную папку **`install`**
